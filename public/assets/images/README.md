# Images

Static images for the game (backgrounds, UI elements, logos, etc.)

## Naming Conventions

- Use lowercase with hyphens: `main-menu-bg.png`
- Group related images with prefixes: `ui-button-play.png`, `ui-button-settings.png`
- Include size in name if multiple versions: `logo-32x32.png`, `logo-64x64.png`

## Recommended Formats

- **PNG**: Best for UI elements, sprites with transparency
- **JPEG**: Best for backgrounds and photos (smaller file size)
- **WebP**: Modern format with good compression (check browser support)

## Example Structure

```
images/
├── backgrounds/
│   ├── menu-bg.png
│   └── game-bg.png
├── ui/
│   ├── button-normal.png
│   ├── button-hover.png
│   └── panel.png
└── logo.png
```
